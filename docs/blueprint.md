# **App Name**: Onboarding Journey

## Core Features:

- Personalized Dashboard: Display employee name, email, and department for a personal welcome.
- Interactive Learning Paths: Guide new employees through curated content based on their department.
- Code Review Tool: Interactive code editor for hands-on learning and skill development.
- Intelligent Code Assistant: A tool to offer suggestions for code improvement, adhering to PEP 8 and best practices. Provides alternative solutions and explanations.
- Progress Tracking: Visually represent the employee's progress through each learning module.
- Gamified Learning Experience: Award points and badges for completing lessons and tasks to encourage engagement.

## Style Guidelines:

- Primary color: Forest Green (#08AA72) to convey growth and learning.
- Background color: Light Grey (#D8DADC) for a clean, professional feel.
- Accent color: Teal (#1FB2B8) to highlight interactive elements and CTAs.
- Font: 'Inter' for body text. Inter is a sans-serif font with a modern, neutral look for easy readability. Use 'Space Grotesk' for Headlines; a proportional sans-serif with a computerized, techy, scientific feel.
- Use Material Icons for a consistent and recognizable visual language.
- Responsive design for optimal viewing on desktop and mobile devices.
- Subtle transitions and animations to enhance user engagement.
